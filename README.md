# CODSOFT Java ATM Interface

## Project Overview

This project is a simple ATM Interface application developed in Java. It simulates basic banking operations through a console-based menu-driven program.

## Features

* Check Account Balance
* Deposit Money
* Withdraw Money
* Input Validation
* Prevents withdrawal when balance is insufficient
* User-friendly menu interface
* Displays transaction status and updated balance

## Technologies Used

* Java
* Scanner Class
* Object-Oriented Programming (OOP)

## How It Works

1. The program starts with an initial account balance.
2. The user selects an option from the ATM menu.
3. The user can:

   * Check Balance
   * Deposit Money
   * Withdraw Money
   * Exit
4. The program validates the input and displays appropriate messages.
5. The menu continues until the user chooses to exit.
